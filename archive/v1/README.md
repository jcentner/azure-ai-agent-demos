# Archived Demos (v1 — Original Azure AI Foundry Agents API)

> **These demos target the original Azure AI Foundry Agents API (v1).** They are preserved here for reference and to accompany the [walkthrough video](../README.md). For updated versions targeting **Agents v2** and the **new Foundry UI**, see the top-level demo directories.

## Demos

| Demo | Description |
|------|-------------|
| [mcp_mslearn_agent](mcp_mslearn_agent/) | Minimal MCP tool demo connecting to the Microsoft Learn MCP server |
| [mcp_local_server_agent](mcp_local_server_agent/) | Three-step demo: local MCP server → MCP Inspector → agent connection with auth |

## Why Archived?

Azure AI Foundry released **Agents v2** with a redesigned API surface and a **new Foundry portal UI**. The v1 demos still work conceptually but reference deprecated SDK patterns and the legacy portal experience. The v2 rewrites in the top-level directories use the current SDK and UI.
